# HyperStudy SaaS
**The most powerful study platform for people with ADHD, ADD, and Dyslexia.**

### Features
- Multi-Account Support
- Smart Quiz from pasted notes
- Live search dropdown
- Full Hebrew support (RTL)
- Pomodoro timer with breaks
- Flashcards, Quizzes, Games
- Embedded YouTube & Khan Academy videos
- Customizable UI themes

### Quick Start
1. Open `index.html` or `app.html`
2. Use any demo account to login

### Tech
HTML + Tailwind + Vanilla JS (Single file ready)

Made with ❤️ for neurodivergent students.
